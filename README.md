# A Python MIDI parser with an additional MIDI player. 
## Features ✨
Parses. That's about it?  
Reads ~6 million events in about 15 seconds. 🦅  
Inefficient (1.5GB for 3 Million notes) ✅



# MIDI Player features
Uses KDMAPI (OmniMIDI synthesizer) for playback. 🎹  
Does relatively well up until 300-500k events/sec (on an i5-13600KF) 🐌  
Channel based visualizer
